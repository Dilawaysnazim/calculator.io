Html

HTML is the fundamental language for creating the structure and content of web pages. It defines the layout and semantic structure of web content using elements like headings, paragraphs, lists, and links

CSS

CSS is used to control the presentation and styling of web content. It allows you to define colors, fonts, layout, and responsive design for your web pages.

JavaScript

JavaScript is a versatile programming language that adds interactivity and dynamic behavior to web pages. It's essential for creating features like sliders, forms, interactive menus, and handling user interactions.

jQuery

jQuery is a JavaScript library that simplifies and streamlines common tasks like DOM manipulation, event handling, and AJAX requests. It's often used to make JavaScript development more efficient
